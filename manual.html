DE LA MODULAR — Web Style Manual

1. Brand Foundation

Tone: Differently serious. Direct. Minimal. Underground. No corporate gloss. No decorative noise.

2. Color System

Primary Background

#0c0c0c

Accent (Voltage Pink)

#f0566d

Panel Surface

#141414

Pink is used sparingly: – Accent lines – Hover borders – Micro highlights Never as background.

3. Typography System

Primary Font

"DIN1451", "Bahnschrift", "Arial Narrow", sans-serif

Usage: – Navigation – Body – Structural headers – UI elements Uppercase preferred for headers.

Experimental Font

"Bevan", serif

Usage: – Hero statements – Taglines – Controlled emphasis Never for body text.

4. Layout System

Max width: 1200px 12-column grid system Generous negative space Sections separated by thin 1px divider line

.grid { display: grid; grid-template-columns: repeat(12, 1fr); gap: 32px; }

5. Motion System

Motion is subtle and restrained. No large parallax. No exaggerated animation.

Scan Drift

Slow vertical movement. Very low opacity. Simulates voltage presence.

Jitter

setInterval(() => { const x = (Math.random() - 0.5) * 0.6; const y = (Math.random() - 0.5) * 0.6; element.style.transform = `translate(${x}px, ${y}px)`; }, 120);

Max displacement: under 1px. Creates tension, not chaos.

6. Interaction Rules

Buttons: – Transparent background – 1px white border – Pink border on hover – No heavy fills

Panels: – Dark surface (#141414) – 1px border – Pink border on hover

Navigation: – Uppercase – Slight letter-spacing – Pink hover state

7. What Not To Do

No gradients. No glass morphism. No drop shadows. No neon glow. No large motion effects. No tech startup aesthetics.

Serious. Controlled. Deliberate.
